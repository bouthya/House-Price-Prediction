# House Price Prediction Using Linear Regression

## Project Overview

This project aims to predict house prices using Linear Regression, a supervised machine learning algorithm. By analyzing various housing features such as area, number of bedrooms, bathrooms, parking spaces, and other property characteristics, the model estimates house prices and helps understand the factors influencing property values.

## Objectives

* Load and explore housing data.
* Clean and preprocess the dataset.
* Perform exploratory data analysis (EDA).
* Select relevant features for prediction.
* Build and train a Linear Regression model.
* Evaluate model performance using regression metrics.
* Visualize actual and predicted house prices.
* Generate insights from the model results.

## Dataset Information

The dataset contains housing-related attributes such as:

* Area
* Bedrooms
* Bathrooms
* Stories
* Parking
* Main Road Access
* Guest Room
* Basement
* Hot Water Heating
* Air Conditioning
* Preferred Area
* Furnishing Status
* Price (Target Variable)

## Project Workflow

### 1. Data Collection

* Load the housing dataset.
* Understand dataset structure and features.

### 2. Data Cleaning

* Check for missing values.
* Remove duplicate records.
* Convert categorical variables into numerical format.

### 3. Exploratory Data Analysis (EDA)

* Analyze house price distribution.
* Study feature relationships.
* Generate correlation heatmaps.
* Visualize key trends.

### 4. Feature Selection

* Identify variables that influence house prices.
* Prepare feature matrix and target variable.

### 5. Model Training

* Split data into training and testing sets.
* Train a Linear Regression model using Scikit-Learn.

### 6. Model Evaluation

Evaluate model performance using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

### 7. Visualization

* Correlation Heatmap
* Area vs Price Scatter Plot
* Actual vs Predicted Price Plot

## Key Findings

* Property area significantly influences house prices.
* Additional amenities such as air conditioning and parking contribute positively to property value.
* Houses located in preferred areas generally have higher prices.
* Linear Regression effectively captures relationships between housing features and prices.

## Business Recommendations

1. Focus on larger properties for higher market value.
2. Improve amenities such as parking and air conditioning to increase property prices.

4. Use predictive models to assist in pricing decisions.
5. Support real estate planning with data-driven insights.

The Linear Regression model successfully predicts house prices based on housing characteristics and provides valuable insights into factors affecting property values.
